== Installation ==
1. Upload the plugin files to the "/wp-content/plugins/plugin-name" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins screen" in WordPress
3. Navigate to the Qbop plugin menu in wordpress to configure.
4. Sign into or create an account on Qbop: www.qbop.com
5. Register your website with qbop if you haven't already done so.
6. Obtain a tracking id from qbop - Account -> My Websites -> click the "Tracking code" option for desired site. On the tracking code page, look for the following code block and copy the tracking id: new qbop.tracker().trackPageView();
7. In the field labeled "Qbop Tracker Id" enter the code you obtained from Qbop.
8. That's it, the tracking code will now be added to the footer section of your site.
